# JianMu

电信数据上传服务端代码，基于[Play framework框架](https://www.playframework.com/)。

## 项目结构

|  目录/文件  |        说明        |
| :---------: | :----------------: |
|    .g8/     |      模板文件      |
|    app/     |        代码        |
|    conf/    |   配置文件和路由   |
|  project/   |    项目配置文件    |
|   public/   |      静态文件      |
|  scripts/   |   部署相关的脚本   |
|    test/    |      测试代码      |
| .travis.yml | Travis CI 部署脚本 |
|  build.sbt  |    项目配置文件    |
